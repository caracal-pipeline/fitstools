Installation
=============
The stable version fitstoolz is available on `PyPI <https://pypi.org/project/fitstoolz>`_ and can be installed using:

.. code-block:: bash

    pip install fitstoolz 

The latest version can be installed directly from github by using:
    
.. code-block:: bash

    pip install git@https://github.com/caracal-pipeline/fitstoolz
